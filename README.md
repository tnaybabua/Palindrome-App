# PalindromCheckerApp
* Use Case 5: Palindrome Check Using Stacks.

* Description:
* This class demonstrates basic palindrome validation
* using creating stacks
* At this stage, the application:
* - Push all char of String to Stack
* - Create a new String names Reversed 
* - pop from stack and add to reversed string
* - Compares characters of both orginal and reversed String
* - Determines whether the string is a palindrome
* - Displays the result on the console
